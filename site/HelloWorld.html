C# by Example: HelloWorld

 
using System;

A simple program that prints "Hello World" to the console.

 
class Program
{
    static void Main()
    {

Console.WriteLine prints a line to standard output

 
        Console.WriteLine("Hello World");
    }
}
 

$ dotnet run
Hello World

Sometimes we'll want to build our programs into binaries. We can do this using `dotnet build`.

 
$ dotnet build
Microsoft (R) Build Engine version [version]
Copyright (C) Microsoft Corporation. All rights reserved.

  Building...
  HelloWorld -> bin/Debug/net[version]/HelloWorld.dll

We can then execute the built binary directly.

 
$ dotnet bin/Debug/net[version]/HelloWorld.dll
Hello World
